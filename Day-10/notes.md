# Day 7 - NLP Basics

NLP = Natural Language Processing

Steps:
- Lowercase
- Tokenization
- Stopword removal

Used in:
- ChatGPT
- Chatbots
- RAG systems

Interview:
NLP helps computers understand human language using preprocessing techniques like tokenization.

Tokenization is required because machine learning models cannot directly process raw text. It converts text into smaller units called tokens, which can then be transformed into numerical representations for model processing.
